## 🦎 Squamata Services
A espinha dorsal técnica do ecossistema, composta por microserviços especializados:
* **Squamata-Login:** Centralizador de autenticação (SSO) utilizando Passport e estratégias modernas de segurança.
* **Squamata-Upload:** Serviço centralizado para manipulação de assets e integração com Cloud Storage (Firebase).
* **Squamata-Gateway** Serviço de centralização de métodos de pagamento para diversos bancos, que se conecta com as plataformas de atendimento e vendas.
